# Datastructures
Notable DataStructure and algorithms implemented in C++
STL with a bit of C++11

Datastructures:
- Graphs
- Binary Trees
- Binary Search Trees
- Priority queue
- Binary Heap

Algorithms:
- Sorting:
  - mergesort
  - quicksort
  - selection sort
  - heapsort
- Graph and Tree:
  - BFS
  - DFS
  - Dijkstra MSP
- Several recursive problems
